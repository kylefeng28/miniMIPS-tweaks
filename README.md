# miniMIPS Tweaks

Prettifies [miniMIPS](http://wwwx.cs.unc.edu/Courses/comp411-002-s17/tools/mips.html) and adds keyboard shortcuts.

Keyboard shortcuts:

```
F1: Assemble
F2: Reset
F3: Step
F4: Run
F5: Memory Dump
F6: Output Trace
```
